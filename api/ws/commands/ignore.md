# ignore

The `ignore` command cancels any debug subscriptions and the log level is restored \(if no other connections are debugging\).

```javascript
{
    "id": 5,
    "cmd": "ignore",
    "sys": "sys-Z6XXA-Kc_v",
    "mod": "mod-Z6XXB1doL4",
    "index": null,
    "name": "ignore"
}
```

responds

```javascript
{
    "id": 5,
    "type": "success"
}
```
